# End-to-End-Waste-Detection-System
This project presents an end-to-end Waste Detection System using computer vision to identify different types of waste materials. The system aims to improve waste management efficiency, support environmental sustainability, and demonstrate how AI-driven solutions can address real-world ecological and societal challenges.
